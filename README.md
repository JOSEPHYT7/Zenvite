# ✉️ Zenvite - Digital Invitations for Every Celebration

**Transform your special moments into unforgettable digital experiences.**

Zenvite is a premium, single-page web application designed to create breathtaking digital invitations. Whether it's a grand wedding, a vibrant Holi party, or a joyful birthday bash, Zenvite brings your celebrations to life with modern aesthetics, interactive animations, and seamless sharing.

## 🛠️ Built with Modern Tech

- **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js
- **Email:** Nodemailer (Gmail SMTP Integration)
- **Animations:** Canvas-confetti, custom CSS micro-animations

## 📦 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/JOSEPHYT7/Zenvite.git
```

### 2. Setup Backend
```bash
cd backend
npm install
# Create a .env file with your credentials (see .env.example)
npm start
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
